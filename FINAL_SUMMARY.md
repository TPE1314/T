## 🚀 快速开始

### 1. 一键启动（推荐）
```bash
# Linux/macOS
./quick_start.sh

# Windows
quick_start.bat
```

### 2. Docker 部署
```bash
# Linux/macOS
./docker_deploy.sh

# Windows
docker_deploy.bat
```

### 3. 手动启动
```bash
# 配置环境
cp env_example.txt .env
# 编辑 .env 文件设置 Bot Token 和管理员 ID

# 安装依赖
pip install -r requirements.txt

# 启动机器人
python start_bot.py --mode polling
```
